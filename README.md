# Aulas TypeScript - Generation Brasil 📚💻

Este repositório contém as atividades e exemplos desenvolvidos durante as aulas de **TypeScript** no **bootcamp de Desenvolvimento FullStack JavaScript da Generation Brasil**.

Durante as aulas, o instrutor apresentava os conceitos e os alunos acompanhavam desenvolvendo os códigos junto com ele.

O objetivo deste repositório é registrar os exercícios realizados durante o processo de aprendizado.

---

## 📚 Conteúdos estudados

Neste repositório estão exemplos e exercícios relacionados a:

- Introdução básica ao TypeScript
- Funções
- Estruturas de dados
- Collections
- Pilha (Stack)
- Fila (Queue)

## ⚙️ Pré-requisitos

Antes de executar os arquivos deste projeto, é necessário ter instalado em sua máquina:

- Node.js
- npm
- TypeScript
- ts-node

---

### Instalar o Node.js

Baixe pelo site oficial:

https://nodejs.org

Após instalar, verifique se a instalação foi concluída corretamente executando no terminal:

```bash
node -v
npm -v
```

Se aparecerem as versões instaladas, significa que o Node.js e o npm estão funcionando corretamente.

---

### Instalar o TypeScript

No terminal, execute:

```bash
npm install -g typescript
```

Depois verifique a instalação:

```bash
tsc -v
```

---

### Instalar o ts-node

O **ts-node** permite executar arquivos TypeScript diretamente no terminal sem precisar compilá-los para JavaScript.

Instale com o comando:

```bash
npm install -g ts-node
```

## 📥 Como clonar o repositório

No terminal, execute:

```bash
git clone https://github.com/BiiaBraga/Aulas-TypeScript-Generation.git
cd Aulas-TypeScript-Generation
```

---

## 📦 Instalação das dependências

Após clonar o repositório, instale as dependências listadas no arquivo `package.json`:

```bash
npm install
```

Esse comando irá instalar automaticamente as bibliotecas utilizadas no projeto, como:

- readline-sync
- @types/node
- @types/readline-sync

---

## ▶️ Como executar os arquivos

Para executar um arquivo TypeScript utilize o **ts-node** no terminal:

```bash
ts-node nomeDoArquivo.ts
```

O resultado da execução será exibido diretamente no terminal.
